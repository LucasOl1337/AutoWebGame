# Triage Labels

As skills usam cinco papeis canonicos. Esta tabela os mapeia para os labels reais do tracker.

| Label canonico | Label no tracker | Significado |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Mantenedor precisa avaliar a issue |
| `needs-info` | `needs-info` | Aguardando mais informacoes |
| `ready-for-agent` | `ready-for-agent` | Especificada e pronta para agente AFK |
| `ready-for-human` | `ready-for-human` | Requer implementacao humana |
| `wontfix` | `wontfix` | Nao sera executada |

Quando uma skill mencionar um papel, use o label correspondente desta tabela.
